# Sales Dashboard – Sara Company

This project is a **Sales Dashboard** for a fictional company named **Sara**. It analyzes sales data and generates visual reports to provide insights into product performance, category trends, and daily sales.

## Features

- Calculate total sales per product and category
- Track daily sales trends
- Generate professional charts:
  - Bar chart for product sales
  - Pie chart for category sales
  - Line chart for daily sales
- Charts saved as PNG files for reporting

## Technologies

- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- CSV for data storage

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/saraissa7/sales_dashboard.git
2. Navigate to the project folder:
cd sales_dashboard
3. Install dependencies:
pip install pandas matplotlib seaborn
4. Run the dashboard:
python main.py

**Charts will be displayed during execution and saved in the project folder as PNG files.**
